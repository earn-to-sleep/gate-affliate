## React NextJS Nextraで即席ブログ作成
WordPressが定番ですが、サーバーとデータベースが必須なので回避。  
Reactを学びたいなと思っていたこともあり、当エロアフィリエイトに辻褄を盛大にオーバーライドして活用してみる。選んだものはNextJSのブログテンプレートNextraBlog。  

```
$ git clone
$ cd <dir>
$ npm install
$ npm run dev
```

で即完成。  
公開先としてサーバーがいらなかったのでホスティングサービスは無料のものを活用できました。VercelというNextJSが運営しているサービスを活用。githubのレポジトリを連携しgitpushすれば即デプロイ。

## Freedomで無料ドメインを取得し、SSLを設定
上記で公開したブログに対し、Freedomで取得した無料ドメインを適用し、Vercelにネームドメインを書き換え、https含む各設定をする。
